# Visual polish update

This update makes the site less plain/white while keeping it professional and recruiter-friendly.

## Files changed/added

- `_includes/head/custom.html` — loads the custom stylesheet through Minimal Mistakes' custom head include.
- `assets/css/custom.css` — adds a soft blue/green background, card styling, shadows, rounded sections, hover states, and page polish.
- `about.md` — converts the About page into a more polished layout with a hero callout, cards, and skill pills.
- `index.html` — keeps the homepage structure aligned with the updated visual style.

## How to apply

Copy these files into your repo, replacing the existing files where needed. Then commit and push:

```bash
git add _includes/head/custom.html assets/css/custom.css about.md index.html
git commit -m "Polish portfolio visual design"
git push
```

If GitHub Pages does not refresh immediately, wait 1–3 minutes and hard refresh the browser.
