# Apply the portfolio updates

These files update the GitHub Pages/Jekyll portfolio for a cleaner recruiter-facing layout and stronger alignment with content workflow architecture, metadata, validation, publishing, developer enablement, and OpenUSD/Omniverse learning.

## What changed

- Replaced the cramped default collection grid with a custom responsive portfolio page.
- Updated the home page positioning from technical writer to senior content workflow architect.
- Updated the About page with stronger content systems language.
- Updated site config title, subtitle, author bio, and GitHub link.
- Added new case studies:
  - Content Workflow Architecture for Reusable Technical Assets
  - Walmart Marketplace API Content Workflow Architecture
  - AWS Security Services Documentation Portfolio
  - Cisco Infrastructure Content Systems
- Expanded existing portfolio pages:
  - OpenUSD & Omniverse Kit Learning Project
  - Claude Code Developer Enablement Guide
  - End-to-End Image Classification with Amazon SageMaker

## How to apply

1. Copy the files and folders from this ZIP into your existing `anusha-jeyaseelan` repo.
2. Replace files when prompted.
3. Commit and push to `main`.
4. Let the GitHub Actions Jekyll workflow finish.
5. Visit: https://anushjeya.github.io/anusha-jeyaseelan/portfolio/

`Update portfolio layout and content workflow case studies`
