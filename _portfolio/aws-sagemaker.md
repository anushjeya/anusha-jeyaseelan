## Overview

This tutorial demonstrates an end-to-end image classification workflow using Amazon SageMaker.

The goal is to help developers understand the full workflow: preparing data, organizing assets, connecting to S3, training a model, deploying the model, and validating the result.

## Tutorial structure

- Prepare the dataset
- Upload and organize data in Amazon S3
- Configure the SageMaker training workflow
- Train an image classification model
- Deploy the model endpoint
- Test predictions and review results

## Documentation approach

The tutorial is structured as executable developer content. It breaks the workflow into repeatable steps, explains prerequisites, and helps users understand where each AWS component fits into the end-to-end process.
